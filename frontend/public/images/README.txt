Use estas pastas para imagens estaticas do frontend.

- `public/images/css/`
  Coloque aqui fotos, texturas, banners e backgrounds usados diretamente no CSS.
  Exemplo de uso: `background-image: url('/images/css/fundo-painel.jpg');`

- `public/images/reports/`
  Coloque aqui logo, brasao ou outras artes institucionais para relatorios.
  Exemplo de nome sugerido: `brasao-5bprv.png`
  Sugestao para uso futuro: configurar esse arquivo em `src/config/reportBranding.js`.

Arquivos dentro de `public/` ficam disponiveis por URL absoluta a partir da raiz do frontend.
